Introduction — 

this program helps students figure out what their final grade will be in any class based on what grades they got on all their assignments



Getting Started —

to run the program, first start by compiling it with the build instructions on the README.md file. then run the program based on what software you're using. 



Step-by-Step Walkthrough — 

1. the program will automatically display good morning/afternoon/evening based on your clock. 
2. the program will prompt you to enter the total points possible. input a whole number greater than 0. 
3. the program will prompt you to enter the minimum points required for A/B/C/D. type a positive whole number. go in descending order (A>B>C>D)

4\. the program will prompt you to enter the assignment score. enter a whole/decimal number 0 or greater. after that it will prompt you to enter the name of the assignment. 



Sentinel Value Explanation — 

when you are done entering assignments, type any negative number. this will trigger the calculation.

Understanding Your Results —

"total points earned" is the sum score of all your assignments. "total points possible" is the total amount of points you can earn in your course (you entered this number earlier). "total percentage" is the your final score given as a percentage. "final letter grade" categorizes your score into what letter it fits into. 



Troubleshooting — 

1. “why is the program crashing or looping weirdly?”

you may have entered letters instead of numbers. scores and cutoffs must be numbers only.



2\. “why won’t the assignment list stop?”

you must enter a negative number (like -1) to end assignment entry.
